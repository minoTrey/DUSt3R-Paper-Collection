# Surveys, Benchmarks & Analysis

## 📚 Overview

This category collects papers that do **not** propose a new reconstruction method. They
survey the field, benchmark existing models against each other, or probe what these
models have actually learned. They are kept separate because the standard paper template
assumes a method with its own results table, and applying that shape to a survey or a
benchmark produces a misleading document.
